# BRModelo

Esta pasta contém os artefatos de modelagem utilizados durante o projeto do banco de dados do NeoLibro.

A modelagem foi elaborada utilizando a ferramenta BRModelo e serviu como base para a implementação física do banco em MySQL.

## Estrutura

brmodelo/

├─ exports

└── bin/

## `bin/`

Contém os arquivos-fonte da modelagem.

Esses arquivos representam o projeto editável do BRModelo e devem ser preservados para futuras alterações estruturais.

Seu papel é equivalente ao de um arquivo de projeto, permitindo reabrir, modificar e evoluir a modelagem do banco de dados.

## `exports/`

Contém exportações da modelagem em formato de imagem (.png), permitindo a visualização da estrutura sem a necessidade de abrir os arquivos-fonte do BRModelo.

## Observação

Os arquivos presentes nesta pasta não são artefatos temporários.

Eles fazem parte da documentação técnica do projeto e devem ser versionados juntamente com o restante da documentação.
