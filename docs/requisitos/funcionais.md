# Requisitos Funcionais - NeoLibro

## RF 01 - Cadastrar obras

O sistema deve permitir a adição de obras literárias.

### Tipos suportados

- HQ;
- Livro;
- Mangá.

## RF 02 - Consultar obras

O sistema deve permitir consultar e visualizar as obras adicionadas.

## RF 03 - Excluir obras

O sistema deve permitir excluir obras cadastradas.

## RF 04 - Atualizar obras

O sistema deve permitir atualizar dados de obras.
