# Regras de Negócio - NeoLibro

## RN 01 - Dados obrigatórios das obras

Toda obra deve possuir:

- Código;
- Título;
- Preço.

### HQ

Toda HQ deve possuir:

- Indústria;
- Edição;
- Volume.

### Livro

Todo Livro deve possuir:

- Autor;
- Editora;
- Ano de publicação;
- Páginas.

### Mangá

Todo Mangá deve possuir:

- Autor;
- Editora;
- Ano de publicação;
- Capítulos.

## RN 02 - Código de rastreabilidade

Toda obra deve possuir código de rastreio.

### Restrições

- O código existe após a adição da obra.
- Não pode haver repetição do código.

## RN 03 - Critério de busca

Ao menos um critério de busca deve ser preenchido.
